WhiteRabbit WebStart

  1. If not yet available, download OpenWebStart from https://openwebstart.com/download and install it.
  2. Download the WhiteRabbit JNLP file.
  3. Start WhiteRabbit by double clicking the whiterabbit.jnlp file. At the first start this will automatically install a new Java Runtime.
  4. For the initial start, you need to confirm that the certificate is correct. Verify the fingerprints:
    • SHA1 fingerprint: 89:EB:AC:59:CC:8E:34:42:BF:1F:22:47:BC:D2:94:C3:7A:04:BD:6C
    • MD5 fingerprint: A0:55:BC:15:77:DA:A3:97:0B:CD:B1:DF:4C:1A:A6:6A
  5. Confirm that the jar file is downloaded from https://whiterabbit.chp1.net/whiterabbitfx-signed.jar.
  6. WhiteRabbit will start now, using configuration file $HOME/.whiterabbit.properties. See the documentation for details.

You can also start WhiteRabbit from the command line: $OPENWEBSTART_DIR/javaws https://whiterabbit.chp1.net/whiterabbit.jnlp

Project page: https://github.com/itsallcode/white-rabbit
